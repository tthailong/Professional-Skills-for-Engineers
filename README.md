# database-systems-assignment2
HCMUT - Database Systems Assignment 2 (MySQL + App)


How to set up:

1. Frontend: run `npm install` (or `npm install --legacy-peer-deps` if React 19 and vaul are incompatible) then `npm run dev`

2. Backend: run `python -m pip install -r requirements.txt` then `uvicorn main:app --reload`
