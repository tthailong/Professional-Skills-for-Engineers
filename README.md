# professional-skills-for-engineers
HCMUT - Professional Skills for Engineers (MySQL + App)


How to set up:

Frontend (using python to run): `cd frontend` then `npm install` (or `npm install --legacy-peer-deps` if React 19 and vaul are incompatible) then `npm run dev`

Backend (using python to run): `cd backend` then `python -m pip install -r requirements.txt` then `uvicorn main:app --reload`

Database (using cmd to run): `cd database` then `mysql -u your_username -p < run_all_mysql.sql`
